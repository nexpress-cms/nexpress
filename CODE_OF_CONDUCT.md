# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

Report concerns to **hahabsw@gmail.com**. Reports are handled confidentially and reviewed within 7 days.

For security vulnerabilities specifically, use [GitHub Security Advisories](https://github.com/nexpress-cms/nexpress/security/advisories/new) instead — see [SECURITY.md](SECURITY.md).
