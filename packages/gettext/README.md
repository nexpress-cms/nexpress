# @nexpress/gettext

Gettext PO translation export/import for
[NexPress](https://github.com/nexpress-cms/nexpress) i18n collections.

The adapter shares live-document routing and validation with
`@nexpress/xliff`, including protected Lexical content and schema-declared
block props.

## Install

```bash
pnpm add @nexpress/gettext
```

## Usage

See the NexPress
[i18n guide](https://github.com/nexpress-cms/nexpress/blob/main/docs/i18n.md).
