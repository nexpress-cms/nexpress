# @nexpress/translation

Format-neutral content translation extraction and application contracts for
[NexPress](https://github.com/nexpress-cms/nexpress).

Most sites should use a concrete interchange adapter such as
`@nexpress/xliff` or `@nexpress/gettext`. Adapter authors use this package so
collection routing, Lexical structure, and block prop validation stay shared.

## Install

```bash
pnpm add @nexpress/translation
```
