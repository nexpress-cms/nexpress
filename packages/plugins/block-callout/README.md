# @nexpress/plugin-block-callout

Adds a translatable info, success, warning, or danger callout to the NexPress
page builder. It also demonstrates a definition-owned pattern, page template,
and English/Korean ICU catalog.

```bash
pnpm exec nexpress plugin add @nexpress/plugin-block-callout
```

Configure tone, title, body, icon visibility, and optional accent color in the
Admin props form. See the
[plugin block guide](https://github.com/nexpress-cms/nexpress/blob/main/docs/plugin-blocks.md).
